Feather Icons for VanJS

Powered by Vite, Vitest & Typescript

Installation

> npm install vanjs-feather
> pnpm install vanjs-feather
> yarn add vanjs-feather

Usage

// import the icon(s)
import { Activity } from "vanjs-feather";

// call anywhere within VanJS codespace
Activity({ class: "icon" })
